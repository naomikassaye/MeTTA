# Personal Learning Knowledge Graph (PLKG)

An AI-powered Knowledge Graph system that helps students visualize and navigate their learning paths. 

## Features
- **Graph-Based Reasoning:** Answers complex questions about prerequisites and learning roadmaps.
- **Natural Language to Cypher:** Uses LangChain and Groq (Llama 3.3) to translate English questions into database queries.
- **Local Graph Storage:** Uses Neo4j for high-performance relationship mapping.
- **Interactive UI:** A clean chat interface built with Streamlit.

## Tech Stack
- **Database:** Neo4j Desktop (Graph Database)
- **LLM** Groq (Llama-3.3-70b-versatile)
- **Orchestration:** LangChain & langchain-neo4j
- **Frontend:** Streamlit
- **Language:** Python 3.11+
